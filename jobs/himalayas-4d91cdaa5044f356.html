Job Description:

We are seeking a highly skilled Senior SQL Server Database Administrator to support database management initiatives. The ideal candidate will possess extensive knowledge and experience in SQL Server administration across Microsoft Azure and on-premises environments. Proven expertise supporting Microsoft Dynamics AX 2012 databases is a plus. This role involves managing all aspects of database administration, including planning, installation, configuration, security, optimization, monitoring, disaster recovery, and performance tuning of SQL Server databases. The candidate should also have strong T-SQL development skills to maintain, optimize and automate processes.

Key Responsibilities:

  • Architect and maintain robust SQL database systems that support application environments, ensuring maximum uptime, performance, and scalability.
  • Design and implement fault-tolerant SQL Server solutions using high-availability patterns in on-premises and Microsoft Azure environments.
  • Design and deploy Microsoft SQL PaaS solutions in Microsoft Azure.
  • Provide SQL Server administration for MS SQL Server versions 2012, 2014, 2016, 2017, 2019 and/or 2022, as well as support for Microsoft Dynamics AX 2012.
  • Plan and implement database backup strategies and perform recovery in case of disaster.
  • Monitor and maintain database storage, health, performance, sustainability, and security.
  • Manage SQL Server high availability, backups, maintenance jobs, encryption, security controls, etc.
  • Analyze SQL query plans, suggest optimizations, and support developers in the SDLC process.
  • Install, configure, and upgrade database management systems and software, including clustering environments.
  • Refine and automate SQLDBA team processes using T-SQL and PowerShell.
  • Collaborate with development teams to optimize database queries and schema design, enhancing application performance.
  • Provide mentorship and guidance to junior DBAs, fostering a culture of knowledge sharing and continuous improvement.

Required Qualifications:

  • Bachelor’s degree in computer science or information technology (advanced degree preferred).
  • Minimum of five (5) years of experience managing and administering enterprise-level SQL databases in SQL Server 2016 or 2017.
  • Minimum of two (2) years of experience designing and deploying Azure SQL Database PaaS solutions in Microsoft Azure.
  • Minimum of five (5) years of experience planning and implementing MS SQL database disaster recovery for enterprise-level applications.
  • Proficiency in physical database design, database changes, enforcement of data integrity, and security management.
  • Strong knowledge of query languages, particularly T-SQL, with the ability to write and optimize complex queries and stored procedures.
  • Familiarity with database replication, clustering, and failover strategies.
  • Ability to troubleshoot and resolve complex database-related issues, collaborating effectively across functional teams.
  • Experience installing, configuring, and patching database management systems and software.
  • Excellent interpersonal skills, initiative, teamwork, and problem-solving abilities.
  • Strong attention to detail, with good oral and written communication skills.
  • Proven experience supporting Microsoft Dynamics AX 2012 databases is a plus.

Preferred Qualifications:

  • Knowledge of PowerShell and database automation scripting.
  • SolarWinds DPA Tool experience, monitoring, and tunning.
  • Certifications such as Microsoft Certified: Azure Database Administrator Associate or similar.

Technical Skills and Relevant Technologies:

  • Advanced knowledge of SQL Server database architecture and administration.
  • Experience with tools such as SQL Profiler, Performance Monitor, SQL Server Management Studio (SSMS), and Azure Data Studio.
  • Strong T-SQL development and query optimization skills.
  • Experience administering SQL Server in both Microsoft Azure and on-premises environments.
  • Experience automating processes and/or support tasks for SQL Server database administration (for example: T-SQL automation, PowerShell, SQL Server Agent jobs, Azure Automation, etc.)
  • Experience applying or implementing AI/ML techniques or tools to automate, monitor, or improve SQL Server database support is a plus.

Soft Skills and Cultural Fit:

  • A positive, collaborative attitude is a must; able to work constructively with colleagues across teams and levels.
  • Exceptional problem-solving skills, with the ability to navigate complex database issues and provide effective solutions.
  • Strong communication skills, capable of conveying technical information to non-technical stakeholders.
  • Ability to work independently while collaborating effectively with distributed teams.
  • A proactive mindset, with a passion for continuous learning and staying updated on database technologies and trends.
