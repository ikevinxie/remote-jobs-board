公司介绍

AIsa成立于2025年，总部位于美国旧金山。公司背后汇聚了顶尖的投资阵容，包括阿里巴巴、Tribe Capital、Draper Associates、Sumitomo Corporation（住友商事）以及Saison Capital等知名机构投资人，是一家专注于 AI 智能体（Agent）基础设施的创业公司，为智能体生态提供统一的 API 网关、大模型智能路由、上下文管理、智能体原生支付协议等资源层服务。我们的客户是智能体开发者和智能体本身。我们的愿景是构建”Amazon for AI Agents” - The Everything Store for AI Agents - 让AI Agents能够更简单地接入、管理和部署 AI 能力。

团队小伙伴是211/985、海外留学及大厂背景，我们是一家小而精的公司

2、开发者关系工程师
关于这个岗位
AIsa 正在构建面向 Agent 时代的基础设施——从统一的资源层（API、LLM、Skill 、Connector）到可控的 Agent 运行时与控制平面，致力于打造一个开发者可以自由上传、组合、计费资源的开放生态。

作为 AIsa 的首位开发者关系工程师，你将是我们与中国开发者社区之间的核心纽带。你需要把复杂的平台能力转化为开发者能轻松上手的内容：清晰的文档、跑得通的示例、有价值的开源项目，以及真实的社区口碑。

这不是一个纯粹写博客的岗位。你是平台采用率的第一责任人，是开发者声音传回产品团队的通道，也是 AIsa 在技术社区的“门面”。我们需要你懂技术、善表达，更要懂开发者。

AIsa 目前处于早期阶段。你不会接手一套成熟的体系，而是要从第一篇文档、第一个示例、第一批开发者开始，亲手搭建起我们的开发者生态。如果你享受“从 0 到 1”的过程，并坚信 Agent 基础设施的未来，这会是一个充满挑战与成就感的位置。

岗位职责

你的核心目标是降低中国开发者使用 AIsa 平台的门槛，建立技术口碑，并形成有效的产品反馈闭环：

• 内容与文档：编写和维护高质量的中文开发者文档、快速上手指南和最佳实践，确保开发者能在最短时间内跑通第一个 Agent，打磨极致的“第一次成功体验”。

• 示例与开源：构建可直接运行的示例项目、SDK 示例和开源工具，用真实代码证明平台能力，并持续运营这些开源资产。

• 社区与布道：活跃于国内主流开发者社区（如 GitHub、掘金、知乎、V2EX、技术微信群等）及线下技术大会（Meetup），进行答疑、分享，建立 AIsa 的技术声誉。

• 反馈闭环：作为开发者在内部的代言人，系统性地收集开发者的痛点和需求，推动 API 设计、文档和开发体验（DX）的迭代。

• 技术验证：以“外部开发者”的视角亲测平台功能，提前发现并解决集成过程中的坑和体验断点。

任职要求
我们寻找的是兼具**“工程师底子 + 表达能力 + 开发者同理心”**的复合型人才：

•技术能力
扎实的工程背景，能独立写代码、跑通 API 集成与 Agent 开发；熟悉 Python；理解 LLM、Agent、RAG 等基本概念。加分项：熟悉 TypeScript / Go；有后端、SDK 或开发者工具开发经验；用过 LangChain 等 Agent 框架。

•内容与表达
能把复杂技术讲简单；具备优秀的中文技术写作能力（文档、教程、博客）。加分项：有公开的技术写作、演讲或视频教程作品；运营过有影响力的个人技术博客/公众号。

•社区与布道
深刻理解国内开发者的痛点与偏好，能用真诚和实用建立信任；有社区运营或技术活动组织经验。加分项：开源项目维护者或活跃贡献者；在技术圈有个人品牌；有 Meetup 或技术大会分享经验。

•产品同理心
具备产品意识，能从开发者视角发现体验断点，并将吐槽转化为结构化反馈；极强自驱力。加分项：有 DevTool/API 平台的 Dev Rel 或开发者体验 (DX) 经验；有早期创业团队经历。
