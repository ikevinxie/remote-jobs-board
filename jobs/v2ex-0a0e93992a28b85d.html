关于我们

Atlas Cloud 是面向全球开发者与企业的全模态 AI API 平台,用一套统一 API 聚合数百个主流大模型(文本、图像、视频、音频)。在稳定的 API 平台业务之上,我们正在从 0 到 1 打造新一代 AI 应用产品——包括 AI 工作流编排、可视化画布、agent 应用、AI 工具页等方向,让用户把多个 AI 模型组合、编排成自己的生成与自动化流程。

你会以这些新产品为主,同时参与官网、模型页、控制台等前端业务的开发。

岗位亮点

  • 远程友好，不强制坐班，适合希望远程工作的前端同学；
  • 报销 AI 工具费用 并鼓励使用 AI 工具提升开发效率；
  • 五险一金，节假日福利；
  • 轻松的办公氛围，压力小；
  • AI 产品方向，能接触大模型、图像/视频生成、开发者平台等场景；

职位描述

  1. 参与 AI 工作流 / 编排 / 画布 / agent 类新产品的前端开发,实现流程编排、可视化交互、结果预览等核心功能;
  2. 对接大模型 API,完成 AI 生成、流式输出等交互场景的落地;
  3. 参与官网、模型页、营销页、开发者控制台等前端业务的开发与迭代;
  4. 负责相关页面的性能优化、体验打磨与线上维护。

职位要求

  1. 熟练掌握 React / Next.js / TypeScript,能独立完成模块开发;
  2. 有较复杂的交互类项目经验——画布、图编辑、拖拽编排、Canvas 等其中一类(用过 React Flow 、tldraw 、Fabric.js 、Konva 等更佳),或有其他复杂前端交互的落地经验;
  3. 对 AI 行业有一定理解:了解当前主流大模型的能力边界与适用场景,清楚不同模态(文本、图像、视频、语音)及不同模型在效果、成本、生成特性上的差异;熟悉主流 AI 产品与工具,能理解模型驱动型产品的业务逻辑;
  4. 了解大模型 API 集成与流式输出(SSE),做过 AI 对话 / 生成类产品者优先;
  5. 掌握常见前端性能优化(虚拟化渲染、Web Worker 、大数据量处理等)与响应式 / 多端适配;
  6. 做事踏实、执行力强,能在方向明确的前提下将功能稳定落地。

加分项

  • 有 AI 应用 / workflow 类产品经验(ComfyUI / Dify / n8n / Coze 等编排类产品的使用或开发);
  • 对 AI 领域保持持续关注,能主动跟进新模型、新工具的能力与应用场景;
  • 熟悉 Canvas / WebGL 渲染优化;
  • 有从 0 到 1 参与项目,或独立完成过完整交互 Demo / 开源项目的经历。
