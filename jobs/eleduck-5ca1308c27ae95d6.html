远程客服专员（居家远程岗）

繁星数据是海外代理IP及人工智能数据基础设施服务商，专注为企业出海、AI大模型场景提供稳定可靠的数据服务，凭借极速响应、7x24小时丝滑体验在代理IP赛道积累了大量开发者与企业客户。现面向全网招募可居家办公的在线技术客服，无需线下到岗，期待细心负责的你加入我们的团队。

岗位职责

  • 客户全链路服务：负责微信客户接待与需求沟通，助完成售后回访、问题跟进闭环，保障客户服务体验。

  • 客户测试跟进与转化： 为意向客户开通测试账号，主动且贴心地跟进客户的测试进展；收集客户在测试中的反馈与诉求，协助提升客户满意度与最终转化率。

  • 销售订单支持：处理客户报价、客户合同初稿等相关行政事务（资料整理、数据核对等）。

  • 基础操作指导与答疑： 协助客户完成平台的基础设置操作，例如：指导客户如何添加 IP 白名单、如何查看账密认证信息、如何生成并复制 API 链接等。

  • 技术问题初筛与工单流转： 成为客户与技术团队之间的桥梁。在客户遇到连接超时、访问拒绝等故障时，能够引导客户提供报错截图或基础信息，对问题进行初步分类，并将复杂的技术故障准确、快速地升级流转给后端技术支持工程师。

  • 服务体验优化： 收集并整理日常工作中的客户高频问题（FAQ），协助团队丰富和完善客服话术库及产品帮助文档。

任职要求

  • 学历基础：本科及以上学历，能够熟练操作Word、Excel等Office常用办公软件，可快速完成数据统计与文档整理工作。

  • 经验偏好：有客服、销售支持、行政相关岗位工作经验者优先，了解互联网基础概念、接触过SaaS类产品服务的候选人会重点考虑。

  • 核心能力：具备良好的沟通表达能力，工作细致耐心、责任心强，能快速适配互联网行业快节奏的工作环境，多线程处理不同客户的需求。

  • 工作时间：9:00-16:00，一周工作六天，全程线上完成工作内容无需线下到岗。

薪酬福利

  • 全面薪酬体系：提供行业内具备竞争力的基础薪资，额外设置绩效奖金、年终奖金，让你和公司共享业务发展成果。

  • 灵活假期安排：可正常享受国家法定节假日、带薪年假等常规假期，针对岗位执行的单休制度，公司同步提供对应的薪资补偿或调休方案，充分保障员工权益。
